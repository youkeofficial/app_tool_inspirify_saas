Documentation API Inspirify (v1)

Base URL : http://localhost:5000/api/v1

🔗 Webhooks (Mode SaaS)

Vous pouvez ajouter un paramètre webhook_url à n'importe quelle requête POST. Une fois la génération terminée, Inspirify enverra une notification POST JSON à cette adresse avec les détails du fichier généré.

1. Génération d'Image

POST /generate/image

Requête

{
  "text": "La vie est belle",
  "author": "Anonyme",
  "webhook_url": "https://votre-site.com/webhook"
}

Réponse

{
  "status": "success",
  "filename": "quote_a1b2c3d4.jpg",
  "url": "/generated/quote_a1b2c3d4.jpg"
}

2. Synthèse Vocale (TTS)

POST /generate/tts

Requête

{ "text": "Bienvenue dans l'aventure SaaS." }

Réponse

{
  "status": "success",
  "filename": "tts_e5f6g7h8.mp3",
  "url": "/generated/tts_e5f6g7h8.mp3"
}

3. Montage Vidéo

POST /generate/video

Requête

{
  "images": ["quote_1.jpg", "quote_2.jpg"],
  "audio": "tts_456.mp3"
}

Réponse

{
  "status": "success",
  "filename": "video_z9y8x7.mp4",
  "url": "/generated/video_z9y8x7.mp4"
}

4. Bibliothèque

GET /library

Réponse

{
  "images": ["quote_1.jpg", "quote_2.jpg"],
  "audios": ["tts_1.mp3"],
  "videos": ["video_1.mp4"]
}

5. Générateur Complet & Animé (Wiki Story)

POST /generate/wiki-story

Outil "All-in-one" : Récupère les données Wikipédia depuis un sujet, découpe le résumé, génère les cartes et le TTS, et applique des effets vidéos d'animations organiques (zoom, fade...).

Requête

{
  "subject": "Leonardo da Vinci"
}

Réponse

{
  "status": "success",
  "filename": "video_wiki_z9y8x7.mp4",
  "url": "/generated/video_wiki_z9y8x7.mp4",
  "text": "Leonardo da Vinci était un polymathe italien..."
}
